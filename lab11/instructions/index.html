Part 2: Conditional Statements and Loops

Recall that an if statement, or conditional statement, is written like so:

if (condition) {
    // Do something that only happens if the condition is true
}

where condition is a conditional expression (i.e. it returns true or false) that determines whether or not the body of the if statement gets executed.

Conditional statements allow us to define multiple states of our webpage. Now, the state of our webpage becomes conditionally dynamic--meaning that the state that one user sees can be completely different from another user based on the decisions of that user.

Also recall that you can create all of these elements using a for loop; recall that for loops are written like so:

for (let variable = start; stoppingCondition; doAfterEachLoop) {
    // Things to do at each iteration
}

where variable is the name of the variable that serves as your iterator (often called i or j by convention--i is short for iterator!), start is the value at which variable should start in the first iteration of the loop, stoppingCondition is a conditional statement that determines when the loop should stop, and doAfterEachLoop is some statement that should be executed at the end of each loop to change the value of variable (e.g. for i to count up, i++, or to count down, i--).
