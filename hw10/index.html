Question 1: Color Buttons

Add code so that the background matches the color of the last clicked button. It should work like the video below.

Recall we discussed the use of element.onclick = someHandlerFunction to let the browser run someHandlerFunction for us when the user clicks on the element. Note that the function, for an element, runs only once after the user finishes clicking.

Also remember you can make use of element.style.someCSSPropertyName = someCSSPropertyValue to dynamically set an element's inline styles. For those CSS property names that include hyphens, for example background-color, you can specify them in camelCase in JavaScript with element.style.backgroundColor = "green".
